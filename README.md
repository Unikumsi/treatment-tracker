# Трекер плана лечения — контекст для продолжения

Интерактивный React-артефакт для отслеживания приёма лекарств. Прогресс
сохраняется в общем хранилище артефакта (`window.storage` с `shared: true`),
чтобы двое людей могли отмечать выполнение с разных устройств.

## Что нужно знать

- **Файл артефакта:** `treatment-tracker.jsx` (React + Tailwind, lucide-react)
- **Старт плана:** 25 мая 2026, длительность 10 дней
- **Хранилище:** `window.storage` с `shared: true` — общий доступ
- **Авто-обновление:** при `visibilitychange` и `focus` подтягивает свежие данные
- **Storage key:** `treatment-tracker-v1`
- **Структура данных:** `{ "<dayIndex 0..9>": { "<medId>": [bool, bool, ...] } }`

## План лечения (расшифровка с рукописной записки)

1. **Промывание глаз** — физ. раствор, от внешнего к внутреннему уголку,
   5×/день, 10 дней, далее постоянно
2. **Данцил** (глазные капли) — по 1 капле, 4×/день, 10 дней
3. **Здак** — 0,1 мл через шприц в пасть, 1×/день, 7–10 дней
4. **Прококолин** (пребиотик) — по 1 мл, 2×/день, 7–10 дней

В трекере все продолжительности унифицированы до 10 дней.

## Возможные доработки

- Указатель «кто отметил» (он / она) на каждой галочке
- Заметки/комментарии по дням
- Push-напоминания (потребует другую архитектуру)
- Версия как чистый HTML с Firebase — если нужна публикация вне claude.ai

## Как опубликовать

В артефакте справа сверху: **Share** → **Publish** → скопировать ссылку.
Второму пользователю нужен бесплатный аккаунт Claude, чтобы открыть ссылку.
